Вкладка "Загрузка оценок" Здесь вы можете легко загрузить оценки в формате .csv или .txt. Просто выберите файл, нажмите кнопку "Загрузить" и удивитесь удобству.
Вкладка "Журнал" Создавайте и редактируйте журнал, добавляйте записи оценок учеников в удобной таблице. Управляйте данными с помощью кнопок удаления и редактирования для каждого ученика.
Вкладка "Статистика (Табличный вид)" Получайте подробную статистику для каждого класса и предмета: средняя оценка, медиана, количество и процент учеников для каждой оценки. А также общая статистика для всех учеников.
Вкладка "Статистика (Графический вид)" Визуализируйте вашу статистику с помощью графиков. Легко анализируйте тренды и распределения оценок для более наглядного представления данных.
